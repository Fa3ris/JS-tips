Basic document flow

Comportement par défaut

Block elements

  • 100% width de l'élément parent
  • height aussi grand que le contenu

Inline elements

  • width height aussi grand que le contenu
  • height aussi grand que le contenu

Block Flow Direction

  • Block elements suivent block flow direction du parent - défini par son writing mode (horizontal-tb : horizontal top-bottom)
  • Chaque block apparaît sur une nouvelle ligne séparée par les marges du block

Inline Flow Direction

  • continue sur la ligne jusqu'à atteindre la width du parent
  • dans ce cas retourne à la ligne

Margin Collapsing

  • Fusion de deux marges adjacentes en la valeur la plus élevée
  • dans ce cas retourne à la ligne

I am a basic block level element. My adjacent block level elements sit on new lines below me.

By default we span 100% of the width of our parent element, and we are as tall as our child content. Our total width and height is our content + padding + border width/height.

We are separated by our margins. Because of margin collapsing, we are separated by the width of one of our margins, not both.
